## 1.遇到的报错：
```
[vite] Dep optimization failed with error:
Could not load path (imported by node_modules/node-sass/lib/extensions.js): ENOENT: no such file or directory, open 'path'
```
解决方法：
```
npm install sass -D
```
以及删除 package.json中多余的 ‘,’号
